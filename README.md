## 🚀 Build Localmente

### 1. Requirements
```bash
# Rust + Cargo (para drones)
# Node.js (para frontend)
# Docker (opcional)
```

### 2. Clonar este repo
```bash
git clone https://github.com/datdanh279-dev/mekong-eco-shield-app.git
```

### 3. Instalación completa
```bash
cd mekong-eco-shield-app/frontend
npm install
cd ../backend
pip install -r requirements.txt
```

### 4. Ejecutar la aplicación

**Backend:**
```bash
cd backend
python main.py
```

**Frontend:**
```bash
cd frontend
npm run dev
```

### 🌐 Aplicación local en http://localhost:3000

## 🌍 Especialista em Gestão de Riscos Climáticos

**-----** **Duración educativa:** 6 a 9 meses (teoría + práctica)

### 📚 Programa de estudos aplicado a impacto social

1. **Fundamentos de Manejo de Riscas Climáticos**
   - Cambio climático, Crecimiento del nivel del mar
   - Análisis de vulnerabilidad, Telemediciones de riesgo
snifão, xâm nhập mặn, nhiệt độ, độ mặn, độ pH

2. **Meteorología avanzada + Sensores IoT**
   - Radar, radares Doppler, vệ tinh nhân tạo
   - Tích hợp cảm biến (nhiệt độ, áp suất, độ ẩm, độ mặn)

3. **Viễn thám bằng máy bay không người lái**
   - Vận hành máy bay không người lái, lập kế hoạch bay tự động
   - LiDAR/GDAL, LiDAR GIS, GIS 3D modeling

4. **Sáng tạo đồ thị và truyền tin**
   - GIS + AI, HyperSpectral data, Redinef (Dự báo Kinh tế)
   - Truyền tin dữ liệu hình ảnh bằng Mesh network

5. **Quyết định khẩn cấp sau thảm họa**
   - Cơ cấu phối hợp liên ngành, tích hợp chuyên gia, kiểm soát thiệt hại bằng AI

### 🚀 Avanzado y Completo

6. **Modelado de Desastres**
   - Modelado de inundación, pronóstico de erosión, evaluación de daño
   - Reducción de riesgo, chuẩn bị thiên thần, kiểm soát thiệt hại

7. **Drones de Control**
   - Máy bay không người lái quan sát, máy bay không người lái vận chuyển, máy bay không người lái cứu viện

8. **Bảo tồn mang tính toàn cầu**
   - Phát thải carbon cao, phát triển bền vững, đa dạng sinh học

## 🌍 Tác động và Mục tiêu

#### 🎓 **Vì một tương lai xanh hơn

- **Sứ mệnh:** Tạo ra một giải pháp toàn diện để cảnh báo sớm, cứu trợ khẩn cấp và thích ứng với biến đổi khí hậu cho các cộng đồng dễ bị tổn thương.

- **Tầm nhìn:** Trở thành hệ thống cảnh báo sớm, dự báo lụt/mặn, cứu hộ tiên tiến hàng đầu khu vực.

#### 📊 **Tác động đến cộng đồng**
- **Cảnh báo trước 24h** thiên tai
- **Lập kế hoạch bay tự động** để kiểm tra vùng bị ảnh hưởng
- **Phương tiện cứu hộ trong thời gian thực** đến các khu vực cách ly
- **Học tập liên tục** để nâng cao dự báo chính xác

#### 🎯 **Mục tiêu giáo dục**
- **Thực hành chuyển giao công nghệ** cho các cộng đồng địa phương
- **Giáo dục dựa trên dữ liệu** giúp hiểu biết khoa học
- **Phát triển bền vững** thông qua một dự án thực tế
